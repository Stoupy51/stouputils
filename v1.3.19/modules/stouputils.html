stouputils package#

A collection of utility modules designed to simplify and enhance the development process.

This package provides various tools and utilities for common development tasks including:

Key Features: - Continuous delivery utilities (GitHub, PyPI) - Display and logging utilities (print) - File and I/O management (io) - Decorators for common patterns - Context managers - Archive and backup tools - Parallel processing helpers - Collection utilities - Doctests utilities
